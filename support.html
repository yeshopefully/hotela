모바일 쿠폰은 어떻게 확인/사용하나요?

쿠폰사용

[모바일 쿠폰 확인 방법]
① 안드로이드(구글 플레이스토어) 또는 iOS(앱스토어)에서 "호텔A" 검색 후 어플 다운로드
② 어플 내 회원 로그인 후 쿠폰함 접속
③ 쿠폰 확인 후 종류별 클릭을 통해 세부정보 확인 가능
[모바일 쿠폰 사용 방법]
① 객실 업그레이드 쿠폰은 기존과 동일하게 객실 예약시점에 쿠폰 번호를 예약실로 전달해주시면 사용 신청이 가능합니다. 체크인 시에는 모바일 쿠폰 화면을 프론트 직원에게 보여주시면 사용 완료 처리를 도와드립니다.
② 그 외, 호텔 지점 별 할인 쿠폰은 선 신청 없이 체크인 시 미리 직원에게 확인해주시면 적용을 도와드립니다.
단, 체크 인 시점에만 적용이 가능하며 체크아웃 시점 또는 체크아웃 후에는 적용이 불가하오니 참고바랍니다.
감사합니다.

객실 업그레이드 쿠폰 사용 방법이 궁금합니다.

쿠폰사용

[객실 쿠폰 사용 방법]
1. 지류 쿠폰 또는 APP 쿠폰함 내 모바일 쿠폰 확인 후 쿠폰 번호를 예약하신 호텔 지점의 객실 예약실로 직접 신청해주십시오.
2. 체크인 시, 모바일 쿠폰 또는 지류 쿠폰을 프론트에 제시해 주시면 사용 완료 처리를 도와드립니다.
모든 혜택은 다른 혜택과 중복 제공되지 않습니다.
※ 쿠폰 사용의 상세한 세부 조건은 이용 약관을 참고 해주십시오.
감사합니다.
